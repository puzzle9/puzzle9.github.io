最好用的 linux 发行版

安装

建议直接去 github 下载最新版本来安装

  • https://github.com/microsoft/WSL/releases

配置

  • https://learn.microsoft.com/zh-cn/windows/wsl/wsl-config

或者用 WSL Settings 进行修改

自定义发行版

下载一个启动器 再搞一个发行版的包 就可以

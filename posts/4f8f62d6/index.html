hexo 搭建一个博客
发布于：2018-01-01 01:00:00 访问：

所谓博客，都是孤芳自赏 nMask’s Blog

所需环境

  • node
  • git
  • github

全局安装 hexo

npm install hexo-cli -g

新建项目

hexo init blog
cd blog
npm install

运行

hexo s

此时打开 http://localhost:4000/ 不出意外是可以看到页面的

插件

  • hexo-admin
  • hexo-blog-encrypt
  • hexo-deployer-git
  • hexo-neat
  • hexo-wordcount
  • hexo-abbrlink
  • Valine
  • hexo-git-backup

主题

  • hexo-theme-indigo

配置 _config.yml

  1. 创建 github 仓库 {name}.github.io

  2. 修改 _config.yml 里 deploy

    deploy:
    type: git
    repository: https://github.com/puzzle9/puzzle9.github.io.git
    branch: master

使用 https 的地址每次 push 可能都需要输入用户名密码
姑且推荐用 git 地址

git@github.com:puzzle9/puzzle9.github.io.git

一片新文章/新页面

hexo new post {PostName}
# `source/_posts/{PostName}`

hexo new page {PageName}
# `source/{PageName}`

生成静态文件

hexo g

将本地文件推送至 github

hexo d

hexo 命令

# 启动服务
hexo server -> hexo s

# 生成静态文件
hexo generate -> hexo g

# 将本地文件推送至 github
hexo deploy -> hexo d

Github-Permission-denied

2018-07-17

问题

使用 git 给 github 推送 时遇到

git@github.com: Permission denied (publickey).

解决方案

一个帮助地址

这个是没通过认证的情况

ssh -T git@github.com

输入 一路回车下去

ssh-keygen

复制 ~/.ssh/id_rsa.pub 内容 到 SSH keys

cat ~/.ssh/id_rsa.pub
